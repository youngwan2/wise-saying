## 프로젝트 개요
- 다양한 주제별 명언/글귀를 간단한 인터페이스를 사용해서 정리하여 볼 수 있는 웹 사이트 입니다. 사용자가 직접 자신의 명언을 등록할 수 있으며, 자신의 명언 혹은 글귀를 이미지로 생성할 수 있는 AI 서비스를 제공하는 것이 특징입니다.
